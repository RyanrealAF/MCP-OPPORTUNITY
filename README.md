# BWB Code Assistant

Industrial-grade ecosystem analyzer and code generation suite.

## Deployment Instructions

This project is configured for Firebase Project: `studio-8574497882-c7183`.

### 1. Configuration
Ensure your `.env` file contains valid Firebase API keys.

### 2. Login to Firebase
```bash
npx firebase login
```

### 3. Deploy
```bash
npm run deploy
```

Note: Genkit AI features require Server-Side execution. Use Firebase App Hosting for the best experience.
